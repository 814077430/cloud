## 常见问题 FAQ

### Q1: RTX 40 系列值得买吗？
**A:** 如果你追求 4K 游戏或需要 AI 创作能力，40 系列的 DLSS 3 和大显存很有价值。如果只是 1080p/2K 游戏，30 系列性价比更高。

### Q2: 显存大小重要还是核心数重要？
**A:** 
- **游戏**: 核心数更重要（影响帧率）
- **AI/3D 渲染**: 显存大小关键（决定能否运行大模型/场景）
- **4K 游戏**: 两者都重要

### Q3: A100 和 H100 怎么选？
**A:** 
- **A100**: 成熟稳定，生态完善，适合大多数 AI 训练/推理
- **H100**: 性能更强（3-4 倍），适合超大模型训练，预算充足选 H100

### Q4: 消费级显卡能用于 AI 训练吗？
**A:** 可以。RTX 4090 (24GB) 适合微调中小模型。但缺乏 ECC 显存、MIG 等功能，不适合生产环境。

### Q5: 多卡互联有必要吗？
**A:** 
- **游戏**: 不需要（SLI 已淘汰）
- **渲染**: 有帮助，线性扩展
- **AI 训练**: 需要 NVLink 高速互联（消费级 40 系不支持）

```
RTX 4090
│   │  └─ 性能等级 (90=旗舰，80=高端，70=中高端，60=主流，50=入门)
│   └──── 代际 (40=40 系，30=30 系)
└──────── 产品线 (RTX=光线追踪)
```

```
H100
│  └─ 性能等级 (数字越大越强)

```
